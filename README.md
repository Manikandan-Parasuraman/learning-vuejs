# learning-vuejs

A hands-on collection of Vue.js exercises and example projects used for learning core Vue concepts from basic bindings and events to lists, computed properties, and dynamic styling.

## Repository structure

- `assignment-*` — small assignment exercises (each folder contains `index.html`, `app.js`, `styles.css`).
- `basics-*` — step-by-step basics lessons and starter code for Vue fundamentals.
- `lists-cond-*` — examples and exercises covering conditional rendering and lists.
- `gs-*` — guided small projects and rebuilding exercises.

Each folder is self-contained; open its `index.html` in a browser to run the example, or serve the root folder with a simple static server.

## How to run

- Open a folder's `index.html` directly in your browser for quick testing.
- Or serve the project with a lightweight HTTP server (recommended for modules or fetch requests):

	- `npx http-server .` (run from the repository root)
	- or `python -m http.server 8080`

## What you'll learn

- Vue instance setup and data binding
- Event handling and methods
- Two-way binding with `v-model`
- Computed properties and watchers
- Conditional rendering and list rendering with `v-if` / `v-for`
- Dynamic classes and inline styles
